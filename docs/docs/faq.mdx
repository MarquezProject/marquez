## How to write Marquez API logs to log file? 

Marquez API is created on the top of [Dropwizard](https://www.dropwizard.io) which provides out of the box support
for different log strategies and contains example on [file logging](https://www.dropwizard.io/en/latest/manual/core.html#file-logging)
which has to be put into `marquez.yml` configuration file.
